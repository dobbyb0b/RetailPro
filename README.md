# RetailPro - Inventory & Billing System

A full-stack web application that digitises inventory tracking and billing for retail shops.

## What It Does
- **Inventory Management**: Add, update, and track product stock levels with low-stock alerts
- **Billing System**: Automated invoice generation with calculations and payment history
- **Sales Reports**: Daily and weekly sales analytics, best-selling items tracking

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Firebase (Realtime Database, Firestore, Authentication)

## Live Project
Built for: Nehru Nagar Minimarket, Vile Parle, Mumbai
Status: In active use

## Key Features
Real-time inventory synchronization
User-friendly interface for non-technical staff
Automated billing with error reduction
Daily sales insights and reporting

## How to Use
1. Clone this repository
2. Open `index.html` in your browser
3. Log in with your Firebase credentials
4. Start managing inventory and billing
